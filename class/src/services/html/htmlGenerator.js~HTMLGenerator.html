    Params:

    NameTypeAttributeDescription
    appConfiguration AppConfiguration

    To read the values of the settings that are going to be send to the file.
    appLogger Logger

    To log messages when the file is generated, when the template is removed, and if it happens, when an error is thrown.
    frontendFs FrontendFs

    To read the contents of the template.
    options HTMLGeneratorOptions
    • optional

    To customize the service.
    valuesService HTMLGeneratorValuesService
    • optional
    • default: null
    • nullable: true

    If specified, instead of getting the values from the app configuration, they'll be retrieved from this service getValues method.

    Throw:

    Error

    if valuesService is specified but it doesn't have a getValues method.
