    public htmlGeneratorCustom(serviceName: string, options: HTMLGeneratorOptions, valuesServiceName: string): Provider source 

    import {htmlGeneratorCustom} from 'jimpex/src/services/html/htmlGenerator.js'

    Generates an HTMLGenerator service provider with customized options and that automatically hooks itself to the after-start event of the app server in order to trigger the generation of the html file when the server starts.

    Params:

    NameTypeAttributeDescription
    serviceName string
    • optional
    • default: 'htmlGenerator'

    The name of the service that will be register into the app.
    options HTMLGeneratorOptions
    • optional
    • default: {}

    Options to customize the service.
    valuesServiceName string
    • optional
    • default: null
    • nullable: true

    The name of a service used to read the values that will be injected in the generated file.

    Provider

    public rootStaticsControllerCustom(files: Array): Controller source 

    import {rootStaticsControllerCustom} from 'jimpex/src/controllers/common/rootStatics.js'

    Generates a controller with an already defined list of files. The controller will get all the files, add route for each one of them and include a middleware provided by the RootStaticsController in order to serve them.

    Params:

    NameTypeAttributeDescription
    files Array

    The list of files. Each item can be a string or an Object with the keys origin for the file route, output for the file location relative to the root, and headers with the file custom headers for the response.

    import {sendFile} from 'jimpex/src/services/common/sendFile.js'

    Generates a function to send files with path relatives to the app executable file is located.

    Params:

    NameTypeAttributeDescription
    pathUtils PathUtils

    To generate the relative paths.

    Return:

    SendFile

    Example:

    // Let's say this is inside an Express middleware.
// Get the function
const send = sendFile(pathUtils);
send(res, 'some-file.html', next);
// If your app is on `/app/index.js`, this will send `/app/some-file.html`.
