normalizeHeaders(headers) → {Object.<string, string>}

Source:
  • services/http/http.js, line 166

It takes a dictionary of headers and normalize the names so each word will start with an upper case character. This is helpful in case you added custom headers and didn't care about the casing, or when copying headers from a server request, in which case they are all tranformed to lower case.
